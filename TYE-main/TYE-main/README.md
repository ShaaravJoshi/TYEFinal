# TYE Web Application

A professional platform for career development and social networking.

## Project Structure

- `src/`: Source code including assets, CSS, and JavaScript.
- `pages/`: HTML pages for the application (Auth, Social, Jobs).
- `pages/home.html`: The main dashboard.

## Getting Started

1. Open `pages/auth/login.html` in your web browser.
2. Login with the following credentials (Mock):
   - **Username**: `admin`
   - **Password**: `password`

## Features

- **Authentication**: secure login flow (mocked for demo).
- **Dashboard**: Central hub for navigation.
- **Social & Jobs**: Placeholder modules for future expansion.

## Development

This project uses standard HTML, CSS, and Vanilla JavaScript. No build step is required.
